HƯỚNG DẪN SETUP SUPABASE - CHI TIẾT TỪNG BƯỚC

=====================================
BƯỚC 1: TẠO USER ADMIN
=====================================
1. Mở https://app.supabase.com
2. Chọn project của bạn
3. Vào Authentication → Users
4. Click "Add user" → "Create new user"
5. Điền thông tin:
   - Email: admin@hoithoxanh.com
   - Password: admin123456
   - Auto Confirm User: ✅ BẬT
6. Click "Create user"

=====================================
BƯỚC 2: CHẠY SCHEMA SQL
=====================================
1. Vào SQL Editor trong Supabase
2. Click "New query"
3. Copy TOÀN BỘ nội dung file: supabase/schema.sql
4. Paste vào SQL Editor
5. Click "Run" (hoặc Ctrl+Enter)
6. Chờ đến khi thấy "Success. No rows returned"

=====================================
BƯỚC 3: CHẠY SEED SQL
=====================================
1. Trong SQL Editor, click "New query" để tạo query mới
2. Copy TOÀN BỘ nội dung file: supabase/seed.sql
3. Paste vào SQL Editor
4. Click "Run"
5. Chờ success message

=====================================
BƯỚC 4: CẤU HÌNH SITE URL
=====================================
1. Vào Settings → Authentication
2. Scroll xu shotgun phần "URL Configuration"
3. Site URL: http://localhost:3000
4. Redirect URLs: 
   - Click "Add URL"
   - Nhập: http://localhost:3000/**
5. Click "Save"

=====================================
BƯỚC 5: KIỂM TRA
=====================================
1. Mở: http://localhost:3000/admin/login
2. Đăng nhập:
   - Email: admin@hoithoxanh.com
   - Password: admin123456
3. Nếu thành công → vào dashboard
4. Nếu lỗi → kiểm tra lại các bước trên

=====================================
LƯU Ý:
=====================================
- File .env.local phải có đúng URL và KEY
- Server phải restart sau khi tạo .env.local
- User admin phải có "Auto Confirm User" = ON
- Site URL phải đúng: http://localhost:3000
